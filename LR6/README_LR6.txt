ЛР6. Краткие пояснения и примеры

Задание 1: просмотр дерева и поиск последовательности
- Скрипт: 6-1.pl
- Запуск (чувствительно к регистру): perl 6-1.pl --root /root/PerlProjects/LR6 --sequence use
- Запуск (без учета регистра): perl 6-1.pl --root /root/PerlProjects/LR6 --sequence USE --insensitive
- Итог: печатает дерево (относительные пути) и отчет с файлами, где найдено совпадение, с количеством вхождений.

Задание 2: замена арабских чисел на римские в файле
- Скрипт: 6-2.pl
- Запуск с выводом в новый файл: perl 6-2.pl --file FileForTask6-2.txt --out out_roman.txt
- Без --out выводит результат в консоль. Конвертируются числа 1..3999, остальные остаются как есть.

Задание 3: замена римских чисел на арабские
- Скрипт: 6-3.pl
- Запуск: perl 6-3.pl --file FileForTask6-3.txt --out out_arabic.txt
- Используется шаблон валидных римских чисел до 3999; остальные подстроки не трогаются.

Задание 4: перекодировка текста
- Скрипт: 6-4.pl
- Основные переходы по условию: win1251 -> koi8-r, koi8-r -> cp866. Также добавлены помощники с utf-8.
- Пример win1251 -> koi8-r: perl 6-4.pl --file input1251.txt --from win1251 --to koi8-r --out out_koi8.txt
- Пример koi8-r -> cp866: perl 6-4.pl --file input_koi8.txt --from koi8-r --to cp866 --out out_cp866.txt
- Для чтения результата в UTF-8 терминале: perl 6-4.pl --file input.txt --from koi8-r --to utf-8 --out readable.txt

Замечание про «кракозябры»
- Если выводите байты cp866/koi8-r прямо в UTF-8 терминал без --out, будут нечитаемые символы. Сохраняйте в файл через --out и открывайте в редакторе с нужной кодировкой или конвертируйте в UTF-8.
