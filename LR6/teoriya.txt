ЛР6. Краткая теория по используемым темам

1. Обход дерева каталогов
- Модуль File::Find рекурсивно обходит каталог, вызывая колбэк для каждого элемента. Параметр no_chdir => 1 позволяет работать с абсолютными путями без смены текущего каталога.
- Для форматирования путей кроссплатформенно применяется File::Spec.
- Регулярные выражения Perl позволяют искать подстроки: флаг i отключает учет регистра. Конструкция qr// создает компилированный шаблон.
- Подсчет вхождений делается через список в скалярном контексте: my $count = () = $text =~ /$re/g;

2. Конвертация чисел арабские ↔ римские
- Римские числа строятся из наборов символов M, D, C, L, X, V, I. Для записи 4 и 9 используются вычитательные формы (IV, IX, XL, XC, CD, CM).
- Алгоритм арабское → римское: идти от больших номиналов к меньшим, вычитая их из числа и добавляя символы к результату.
- Алгоритм римское → арабское: проход по символам слева направо, если текущий меньше следующего — вычитаем, иначе прибавляем.
- Регулярное выражение валидного римского числа до 3999: (?i:M{0,3}(?:CM|CD|D?C{0,3})(?:XC|XL|L?X{0,3})(?:IX|IV|V?I{0,3}))

3. Работа с файлами
- Для корректной обработки произвольного содержимого файлы читаются/пишутся в бинарном режиме (binmode :raw), чтобы не ломать байты и окончания строк.
- Аргументы командной строки удобно разбирать через Getopt::Long (ключи с длинными и короткими именами).

4. Перекодировка текста
- Модуль Encode предоставляет decode (байты → Unicode-строки) и encode (Unicode → байты в нужной кодировке).
- В лабораторной используются переходы Windows-1251 → KOI8-R и KOI8-R → CP866 (DOS). Важно выбрать правильные метки кодировок для Encode: windows-1251, koi8-r, cp866.
- Проверка допустимых направлений перекодировки защищает от некорректного вызова.

5. Общие рекомендации
- Всегда проверяйте существование файлов и каталогов перед обработкой.
- В текстовых заменах используйте \Q...\E для экранирования пользовательского ввода в регулярных выражениях.
- При необходимости взаимодействия с пользователем добавляйте подсказки и примеры запуска в вывод программы.
